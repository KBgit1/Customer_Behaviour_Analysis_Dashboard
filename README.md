# Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior to identify purchasing patterns, customer segments, and revenue insights. The analysis combines **SQL querying**, **Python-based data analysis**, and **Power BI dashboards** to transform raw transactional data into meaningful business insights.

The objective is to demonstrate the complete **data analytics workflow**, from data extraction and analysis to visualization and decision support.

---

## Objectives

The project focuses on answering key business questions such as:

* What is the total revenue generated by different customer demographics?
* Which products have the highest average customer ratings?
* Do subscribed customers spend more than non-subscribers?
* How do shipping methods affect purchase behavior?
* Which products are most frequently purchased within each category?
* How do discounts influence purchasing patterns?
* What is the revenue contribution of different age groups?

---


## Project Workflow

### 1. Data Exploration (SQL)

SQL queries were used to extract insights from the dataset including:

* Revenue analysis by gender
* High spending customers using discounts
* Top rated products
* Customer segmentation based on purchase history
* Subscription behavior analysis
* Product popularity within categories


### 2. Data Analysis (Python)

Using **JupyterLab**, the dataset was analyzed using Python libraries.

Key tasks performed:

* Data cleaning
* Exploratory Data Analysis (EDA)
* Distribution analysis
* Correlation analysis
* Customer behavior visualization

Libraries used:

* Pandas for data manipulation
* NumPy for numerical operations
* Matplotlib for plotting
* Seaborn for advanced visualizations



### 3. Data Visualization (Power BI)

An interactive **Power BI dashboard** was built to visualize key insights.

Dashboard features:

* Revenue breakdown by category
* Customer demographics analysis
* Top products by sales
* Subscription vs non-subscription spending
* Discount impact on purchasing behavior

The dashboard allows interactive filtering to explore patterns across different customer segments.
